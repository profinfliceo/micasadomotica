Simulamos nuestro sistema de control

Para simular nuestro sistema de control utilizaremos una plataforma on line llamada TINKERCAD, que posteriormente también la utilizaremos para programar nuestros sistema.

Web de la plataforma TinkerCad

Uso de Arduino en Tinkercad

Montemos nuestro circuito en Tinkercad

Elementos que vamos a utilizar en nuestro montaje:

  1.  Placa Arduino UNO.
  2.  Resistencias 220 Ohomios (3) y resistencia de 4k7 ohmios (1).
  3.  Diodos led rojos (2).
  4.  Sensor LDR (1).
  5.  Zumbador (1).
  6.  Sensor de ultrasonido (1).
  7.  sensor de gases (1).
  8.  Cables de conexión (Jumpers)
  9.  Una o dos placas protoboard.

Vamos a realizar el montaje y simulación de nuestro sistema de control en el Aula de Informática, utilizando la plataforma TinkerCad.
