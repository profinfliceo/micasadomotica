Tipos

Sistema de control de lazo abierto

 

En este tipo de sistemas NO se tiene en cuenta la señal de salida en la acción de control.

Sistema de control de lazo cerrado

 

En este tipo de sistemas SI se tiene en cuenta la señal de salida en la acción de control.

Esta señal de salida se emplea para ajustar la orden que el controlador da al proceso para que se aproxime a un valor preestablecido.

ELEMENTOS QUE INTERVIENEN

o Sensor: Mide la magnitud deseada

o Comparador: Compara la magnitud medida con el valor establecido

o Controlador: Procesa la información y emite las órdenes

o Actuador: Ejecuta las órdenes del controlador
